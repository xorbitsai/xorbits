=======
Xorbits
=======

Xorbits is a tensor-based unified framework for large-scale data computation
which scales numpy, pandas, scikit-learn and many other libraries.

Getting Started
---------------

Starting a new runtime locally via:

.. code-block:: python

    >>> import xorbits
    >>> xorbits.init()

Or connecting to an existing Xorbits cluster:

.. code-block:: python

    >>> import xorbits
    >>> xorbits.init(address="http://<ip>:<port>")
